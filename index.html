Predator Prey Game

 This is a computer playable 
  version of the camp game/game
  theory problem posed by Conway.
  In the game, you play as a 
  frog. You are trying to get
  the bug.

Bug AI rules:

  • The bug moves away from the predator if the distance to you < 160.
  • The bug moves toward the marsh for food if the distance to you > 160.
  • The bug dies if you are centered with it.

Keys:

  • Left Arrow
  • Right Arrow
  • Up Arrow
  • Down Arrow
